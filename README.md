## 组件介绍
* NFSimpleScrollRect
    * 提供简单的滑动到目标的功能. 
      Provide simle scroll to target function.
    * 当可以滑动到中间的时候，目标会居中，当无法在拖动的时候，会留在边缘.
      When it has enough distance to be in the middle of view, it will be. If not, it will be in the edge.
